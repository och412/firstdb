Built in function

  • abs() : 절대값
  • min() : 최솟값
  • max() : 최댓값
  • dir() : 매서드 확인
  • id() : 메모리 주소 확인
  • sorted() : 정렬
  • reverse() : 역순 %역정렬이 아님

  • enumerate() : 순서를 매김 (for문 하면서 설명할 예정)

  • 형변환 :

    • set() : 집합으로 형변환
    • dict() : 딕셔너리로 형변환
    • int() : 불리언으로 형변환
    • bool() ; 불리언으로 형변환
    • int() : 정수형으로 형변환
    • str() : 문자열으로 형변환
    • list() : 리스트로 형변환

  • ord() : 문자를 숫자로 변환

  • chr() : 숫자를 문자로 변환
  • sum() : 전부 더함(더할수 없으면 에러)
  • filter() : 필터링(수학점수 90점 이상인 사람은? 이런식)

  • print() : 출력

  • format() : 콤마찍기

  • len() : 길이출력!! 중요!!!!!!!!
  • type() : 타입 알아보기 중요!!
  • range() : 연속된 값 생성(0~99 -> range(100)이라는 의미임. 100을 포함하지 않음)
  • zip('abc', 'def')
  • map(lambda x:x**2, [1,2,3])
  • round() : 소수점 반올림
